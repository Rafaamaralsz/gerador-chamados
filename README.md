# Gerador de Chamados

Projeto Angular simples para gerar descrições padronizadas de chamados técnicos.

## Requisitos

- Node.js 24.15 ou superior dentro da linha 24 LTS
- npm

## Como executar

```bash
npm install
npm start
```

Depois, abra `http://localhost:4200`.

## Funcionalidades

- Seleção da empresa
- Campo livre para condomínio ou departamento
- Categoria do problema
- Descrição livre do problema
- Checklist de procedimentos realizados
- Geração automática do texto
- Edição manual do resultado
- Copiar texto
- Limpar formulário

Os dados das empresas são fictícios e podem ser alterados em `src/app/app.component.ts`.
